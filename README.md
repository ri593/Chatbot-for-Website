# AIC-RNTU Chatbot

A modern, responsive, and interactive floating chatbot widget for the **AIC-RNTU Foundation** website. Built with pure HTML, CSS, and JavaScript.

![Chatbot Preview](https://via.placeholder.com/800x400/003087/FFFFFF?text=AIC-RNTU+Chatbot+Preview)

---

## ✨ Features

- **Floating Chat Bubble** with smooth hover animations
- **Fully Responsive** chatbot window (380px × 520px)
- **Beautiful UI** with gradient themes and modern design
- **Quick Reply Buttons** (Chips) for easy navigation
- **Multi-level Menu System** (Main Menu → Sub Menu)
- **Typing Indicator** simulation
- **Auto Timestamps** on every message
- **Smooth Animations** and transitions
- **Predefined Responses** for all major sections of AIC-RNTU
- **Mobile-friendly** design
- **No external dependencies** (except Font Awesome)

---

## 🎯 Sections Covered

- About AIC-RNTU (Who We Are, Vision, Impact)
- Services & Offerings
- Facilities (Fab Lab, IoT Lab, Co-working, etc.)
- Programs (Ideate, Discover, Growth, Investment Bridge, IR Program)
- Funding Schemes (SISFS, NIDHI-SSP, MSME Hackathon, iDEX)
- Mentors & Team
- Incubated & Funded Startups
- Investment Partners
- Application & Partnership Forms
- Contact Information
